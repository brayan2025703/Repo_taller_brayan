# Portafolio Web - Repo Taller Brayan

## Descripción
Este proyecto consiste en la creación de un portafolio web utilizando HTML y CSS, desarrollado en Visual Studio Code y publicado con GitHub Pages.

## Contenido del proyecto

- index.html → Estructura del portafolio
- style.css → Diseño de la página
- assets/ → Carpeta de recursos (imágenes)
- README.md → Documentación

## Información incluida en el portafolio

- Foto personal
- Descripción
- Conocimientos
- Estudios
- Hobbies
- Sueños

## Tecnologías utilizadas

- HTML5
- CSS3
- Git
- GitHub
- Visual Studio Code

## Flujo de trabajo

1. Creación del repositorio
2. Desarrollo del proyecto en Visual Studio Code
3. Uso de git add, commit y push
4. Publicación con GitHub Pages
